# Care+ — Arabic translation glossary

Canonical Arabic equivalents for recurring product, clinical, and UI
terms. Translator and reviewer both work from this list. When two
acceptable translations exist (e.g. shift), pick one and stick with it.

Register: **Modern Standard Arabic (MSA / فُصحى)** — no regional dialect.

## Brand

- Brand wordmark "Care+" stays Latin in the logo and in any in-page product reference.
- Arabic body copy refers to it as `Care+` with no transliteration.

## Hard rules

- No regional dialect. MSA only.
- Medical drug names: pharmacist judgement — common Latin-script names stay Latin.
- User-generated content is never translated (nurse bios, donation post bodies, booking notes, observations).
